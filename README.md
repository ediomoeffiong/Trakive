# Trakive

> Internship Onboarding & Performance Tracking Platform

Trakive is an intern onboarding and performance tracking platform designed to help interns stay organized, complete assigned tasks, and monitor their progress throughout their internship. It gives interns a single place to see what they need to do, what they have completed, and how they are performing over time.

## Repository Structure

```
trakive/
├── frontend/    ← React + Vite application (active development)
├── backend/     ← Node.js / Express API (planned)
└── README.md
```

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

The app will be available at **http://localhost:5173 (Development) / https://trakive.vercel.app (Production)**

### Backend

Backend development has not started. See `backend/README.md` for the planned stack.
