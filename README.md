# Trakive

> Internship Onboarding & Performance Tracking Platform

## Repository Structure

```
trakive/
├── frontend/    ← React + Vite application (active development)
├── backend/     ← Node.js / Express API (planned)
└── README.md
```

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

The app will be available at **http://localhost:5173**

### Backend

Backend development has not started. See `backend/README.md` for the planned stack.

---

Built with React, Vite, TailwindCSS, Zustand, and Framer Motion.
